A minimal React (Vite) landing page template for Centive with a dark theme and purple accents.

Palette
- Background / primary: #000000
- Dark gray: #181818
- Primary accent (purple): #7C3AED
- Secondary accent: #5E5D5D
- Primary text: #FFFFFF
- Secondary text: #9CA3AF

Sections included
- Hero (big headline, subheadline, email waitlist)
- How it works (4 steps with imagery)
- FAQs (6 entries)
- Join (first-access form)
- Footer (contact & boilerplate)

How to run
1. Install dependencies:
   - npm install
2. Dev:
   - npm run dev
3. Build:
   - npm run build
4. Preview build:
   - npm run preview

Notes / next steps
- Replace placeholder images with your app screenshots.
- Connect the email forms to your backend(commented where to add).

This is intentionally lightweight and easy to extend.
